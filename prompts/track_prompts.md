# Track-Specific Prompts

## THLP - Pattern Learning

```
This is a Pattern Learning question (THLP).

Focus on:
- Identifying patterns in the given examples
- Learning rules from the pattern structure
- Applying learned rules to new cases
- Recognizing pattern exceptions

Strategy:
1. Examine all examples carefully
2. Identify the underlying rule or transformation
3. Test the rule against each example
4. Apply the rule to select the correct answer
```

## TTM - Metacognitive Calibration

```
This is a Metacognitive Calibration question (TTM).

Focus on:
- Calibration of confidence with actual knowledge
- Detecting when you might be uncertain
- Avoiding overconfidence on ambiguous problems
- Self-assessment of error likelihood

Strategy:
1. Assess how certain you are of the answer
2. Consider sources of uncertainty
3. Rate confidence honestly (not default to high)
4. Be explicit in reasoning about any doubts
```

## TAGP - Attention Control

```
This is an Attention Control question (TAGP).

Focus on:
- Selective attention to relevant features
- Sustained focus on target patterns
- Attention shifting when context changes
- Filtering out distracting information

Strategy:
1. Identify the attentional target
2. Selectively process relevant information
3. Maintain focus through distractors
4. Shift attention appropriately when required
```

## TEFB - Executive Functions

```
This is an Executive Function question (TEFB).

Focus on:
- Multi-step planning and sequencing
- Working memory maintenance
- Cognitive flexibility and adaptation
- Error detection and correction

Strategy:
1. Break down the problem into steps
2. Plan the sequence of actions
3. Maintain relevant information in working memory
4. Be ready to adapt if initial plan fails
```

## TSCP - Social Cognition

```
This is a Social Cognition question (TSCP).

Focus on:
- Theory of Mind (attributing mental states)
- Pragmatic inference from social context
- Understanding social norms and conventions
- Interpreting indirect and implied meaning

Strategy:
1. Consider the social context
2. Infer mental states of involved parties
3. Apply pragmatic understanding of intent
4. Use social norms to interpret ambiguous cases
```
